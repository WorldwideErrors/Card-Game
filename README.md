# Card Game

Voor het leren van de basis van Swift heb ik op basis van een tutorial van Youtube een kaartspel gemaakt. 
Het idee van het kaartspel is dat je een kaart trekt, je kan een hogere kaart hebben of een lagere kaart. 
Als je een hogere kaart hebt krijg je een punt, als de CPU een hogere kaart heeft krijgt de CPU een punt.

De playlist die ik gevolgd heb vind je [hier](https://www.youtube.com/playlist?list=PLMRqhzcHGw1ZHtM5xYcZbJ8oUZ0aVTasI).

---

## 🚀 Features
- Een kaart trekken
- Scores op basis van welke kaart je krijgt
- Een UI die past bij het spel

---

## 🛠️ Tech Stack

- **Language:** Swift
- **UI:** SwiftUI

---

## 📦 Installation

### Requirements

- Xcode: 15+
- Swift: 5.9+
- iOS Deployment Target: 15.0+

### Clone project

```bash
git clone https://github.com/WorldwideErrors/Card-Game.git
cd Card-Game
```

### Open the Project
```
Open het .xcodeproj of .xcworkspace bestand in Xcode:
open Card-Game.xcodeproj
```

## Run the app
```
Select a simulator or your physical device
Press ⌘ + R to build and run
```
